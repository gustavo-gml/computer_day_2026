# Regras do Projeto: Computer Day - IFSULDEMINAS

## Contexto
- Site institucional para evento de computação.
- Objetivo atual: Criar um "molde" estrutural (Skeleton) antes da definição da identidade visual.

## Regras de Desenvolvimento
- **Estilo:** Use apenas Tailwind CSS.
- **Cores:** Use variáveis do Tailwind (ex: `bg-primary`, `text-secondary`). Por enquanto, defina essas cores como tons de cinza no arquivo de configuração.
- **Componentização:** Cada seção das fotos de referência (Hero, Sobre, Agenda, Contato, Mapa) deve ser um componente separado em `src/components/`.
- **Linguagem:** O código deve ser em TypeScript (.tsx).
- **Imagens:** Use placeholders para logos e fotos por enquanto.
